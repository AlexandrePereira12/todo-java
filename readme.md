# Todo Java

Projeto Java com Spring Boot para gerenciamento de tarefas (todo list). Este repositório contém uma aplicação backend capaz de criar, listar, atualizar e excluir tarefas por meio de uma API REST.

## O que é

Este projeto é uma aplicação de exemplo em Java utilizando o framework Spring Boot. Ele serve como um sistema simples de gerenciamento de tarefas, ideal para testar conceitos de desenvolvimento web em Java, APIs REST e persistência de dados.

## Requisitos

- Java 17 ou superior
- Maven Instalado

## Como rodar

1. Abra o terminal na pasta do projeto.
2. Como o projeto usa Maven e tem um arquivo `pom.xml`, execute:

   ```bash
   mvn spring-boot:run
   ```


3. Se o projeto usar Gradle e houver um arquivo `build.gradle`, execute:

   ```bash
   ./gradlew bootRun
   ```

4. Após iniciado, a aplicação normalmente ficará disponível em `http://localhost:8080`.

## O que fazer

- Criar tarefas
- Listar tarefas existentes
- Atualizar o status ou dados de tarefas
- Excluir tarefas
